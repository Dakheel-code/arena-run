Arena Run implements a comprehensive role-based access control (RBAC) system with four distinct roles: Super Admin, Admin, Editor, and Member. Each role has specific permissions designed to maintain security while enabling collaboration.

Super Admin
Video Management
  • View all videos (published & unpublished)
  • Upload new videos
  • Edit all videos
  • Delete any video
  • Publish/unpublish videos
User Management
  • View all members
  • Manage member accounts
  • Assign any role (including Super Admin)
  • Delete user accounts
  • View member profiles
Analytics & Logs
  • View all statistics
  • Access login logs
  • View session history
  • Monitor user activity
System Settings
  • Manage system settings
  • Configure notifications
  • Manage Discord integration
  • Full admin panel access
Admin
Video Management
  • View all videos (published & unpublished)
  • Upload new videos
  • Edit all videos
  • Delete any video
  • Publish/unpublish videos
User Management
  • View all members
  • Manage member accounts
  • Assign roles (up to Admin)
  • Cannot assign Super Admin role
  • Cannot delete accounts
Analytics & Logs
  • View all statistics
  • Access login logs
  • View session history
  • Monitor user activity
System Settings
  • Manage system settings
  • Configure notifications
  • Full admin panel access
Editor
Video Management
  • View all videos (published & unpublished)
  • Upload new videos
  • Edit all videos
  • Publish/unpublish videos
  • Cannot delete videos
User Management
  • Cannot view members list
  • Cannot manage accounts
  • Cannot assign roles
Analytics & Logs
  • View content statistics
  • Cannot access login logs
  • Cannot view session history
System Settings
  • Limited admin panel access
  • Cannot manage system settings
Member
Video Access
  • View published videos only
  • Watch videos with watermark
  • Cannot upload videos
  • Cannot edit videos
  • Cannot delete videos
User Management
  • No access to members list
  • Cannot manage accounts
  • Cannot assign roles
Analytics & Logs
  • No access to statistics
  • No access to logs
System Settings
  • No admin panel access
  • No system settings access
